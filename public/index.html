Online Shopping API

Overview

Welcome to our Online Shopping API. This API provides a comprehensive set of endpoints for managing an e-commerce platform, including user authentication, product management, order processing, and more.

API Collections

Admins

Endpoints for admin users to manage user resources. Operations include retrieving all users, creating new users, and deleting existing users.

Login for Testing:
Email: admin@gmail.com
Password: admin

User

User Authentication & Account Management API for managing user data, orders, complaints, notifications, and account lifecycle.

Products

Manage products with operations like retrieving all products, deleting products, getting a product by ID, creating new products, and updating existing products.

Complaints

Allow users to submit complaints for admin review. Includes routes for retrieving, creating, updating, and deleting complaints.

Cart

Manage customer shopping carts, including adding products, retrieving the cart, and removing products. Authentication required.

Order

Manage the entire order lifecycle for customers, admins, and delivery personnel. Supports creating, updating, canceling orders, and confirming deliveries.

Vendors

Retrieve vendor-specific data and access delivered order items for admins and related vendors.
